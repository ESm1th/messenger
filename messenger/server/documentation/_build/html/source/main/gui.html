class gui.FormFactory(*ars, **kwargs)¶

Custom form layout for simplefied form creation process. Accepts in constructor or sets ‘field’ names and then creates rows. Fields passed as ‘fields=[…]’. Number of rows - len(fields). Format of row: <field> QLabel - <field> TitledLineEdit.
